# PAUDA (accessibilite.data.gouv.fr)

## Le projet

PAUDA est le Point d'Accès Unique aux Données d'Accessibilité. Il a pour vision de faciliter l'accès, l'ouverture et la réutilisation des données d'accessibilité.

PAUDA se veut :
- Un guichet d’entrée pour les recherches de
  - Données d’accessibilité, qui renverra vers les bases de données existantes (AccesLibre, transport.data.gouv.fr, data tourisme, etc.)
  - Documentation sur les normes / standards existants
- Un outil de visualisation, voire de modélisation / statistiques sur les données référencées
- Un lieu de dépôt des données sans « maison » (ex. celles des associations)
- Et surtout une équipe qui facilitera
  - L’ouverture et la réutilisation des données
  - La normalisation des données en y participant, voire en l’initiant

## Ce répertoire

Il s'agit principalement d'un répertoire dédié à la gestion du produit PAUDA dans lequel l'ensemble des parties prenantes peut partager :
- Ses questions et remarques
- Les tâches en cours
- Les problèmes à traiter

La visualisation de la gestionn du projet PAUDA est faite via [ce projet GitHub](https://github.com/orgs/opendatateam/projects/6).
